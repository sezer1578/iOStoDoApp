# iOStoDoApp
I developed To Do App with Swift and Sqlite for iOS. User can add a new task and delete or update their tasks. Also, user can search ther tasks. 

### Technologies
- [√] VIPER Pattern
- [√] SQLite
- [√] Tableview
- [√] Navigation
- [√] Auto Layout
- [√] Dark Mode
- [√] Launch Screen
